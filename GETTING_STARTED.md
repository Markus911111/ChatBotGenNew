# Getting Started

Welcome to the ChatBotGenNew project! This guide will help you integrate the chatbot into your system. Follow these steps to get started:

## Prerequisites
1. Ensure that you have the following software installed on your machine:
   - [Node.js](https://nodejs.org/)
   - [Git](https://git-scm.com/)

## Step 1: Clone the Repository
Open your terminal and run the following command:
```bash
git clone https://github.com/Markus911111/ChatBotGenNew.git
```

## Step 2: Navigate to the Project Directory
Change into the project directory:
```bash
cd ChatBotGenNew
```

## Step 3: Install Dependencies
Run the following command to install the required dependencies:
```bash
npm install
```

## Step 4: Configure the Environment
Create a `.env` file in the root of the project and fill it with the necessary configuration parameters. You can find a sample `.env.example` for reference.

## Step 5: Run the Application
To start the application, use the following command:
```bash
npm start
```

## Conclusion
You should now see the chatbot running! For further customization and advanced usage, check the documentation within the repository.
